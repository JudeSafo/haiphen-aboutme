Launch the interactive shell with haiphen -i and follow the guided prompts below.

1
Onboarding 5 steps · Free tier · No prerequisites
  1. Welcome & Orientation

    See what Haiphen offers, compare plan tiers (Free, Pro, Enterprise), and get links to documentation.

  2. Login via Browser

    Authenticate with GitHub or Google. Your browser opens, you approve access, and the CLI receives your session automatically. Skipped if you're already logged in.

  3. Account Status

    View your current plan, active services, and account details at a glance.

  4. Platform Health Check

    All six intelligence services are pinged. You see a live status grid confirming each endpoint is responding.

  5. Choose Next Workflow

    Pick where to go next: Start Trading or Explore Prospect Intelligence. The shell transitions directly into your chosen workflow.

2
Trading 8 steps · Pro plan · Sandbox-first
  1. Connect Broker

    Link your brokerage account. Start with Alpaca paper trading to test safely before going live. Skipped if already connected.

  2. Account Summary

    View your portfolio value, buying power, and current positions in a compact dashboard.

  3. Safety Limits

    Set maximum order size, daily loss limits, and position caps. Your limits persist across sessions.

  4. Place a Trade

    Execute a trade through the guided interface. Select symbol, quantity, and side. Large orders trigger an extra confirmation prompt.

  5. Order History

    Review your recent orders with status, fill price, and timestamps in a compact table.

  6. Live Updates

    Stream real-time position changes and market data via WebSocket connection.

  7. Signal Rules

    Configure automated entry and exit conditions using YAML-based signal rules.

  8. Signal Daemon

    Start background monitoring that watches markets and fires notifications when your signal rules trigger. Skipped if already running.

Paper Trading Only — The interactive shell defaults to sandbox mode. No real capital is at risk during the guided walkthrough.
3
Prospect Intelligence 7 steps · Free to browse · Pro to investigate
  1. API Credentials

    See which data source keys are configured (NVD, GitHub Advisory, Shodan). Enterprise users can set missing credentials here.

  2. Browse Targets

    Filter Fortune 500 companies by sector (Financials, IT, Health Care, etc.) and select a target for analysis.

  3. Intelligence Scan

    Scan your selected target across vulnerability databases, regulatory filings, and infrastructure signals. See how many leads are discovered.

  4. View Leads

    Browse discovered leads in a table sorted by severity, with signal type, entity name, and CVSS score. Select one for deeper analysis.

  5. Run Investigation

    Run the 6-service analysis pipeline on a lead. See aggregate risk score, per-service breakdown, synthesis summary, and data gaps. Requires Pro plan.

  6. Generate Report

    Export a professional LaTeX report with executive summary, threat landscape, impact assessment, and recommendations. Optionally compile to PDF. Requires Pro plan.

  7. Send Advisory

    Draft and preview an infrastructure advisory email for the target organization. Review subject line and body before sending. Enterprise only.
