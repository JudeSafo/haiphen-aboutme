CLI Installation

Install the Haiphen CLI on any platform and authenticate in seconds.

macOS (Homebrew) brew install haiphen/tap/haiphen
Linux / macOS (curl) curl -fsSL https://get.haiphen.io | sh
Windows (Scoop) scoop install haiphen
Docker docker pull ghcr.io/haiphen/cli:latest
haiphen login haiphen status haiphen serve

Intelligence Services

Six production-ready analysis engines, each with a CLI command and demo.

Haiphen Secure Vulnerability & compliance scanning haiphen secure scan --target <target>
Network Trace Protocol-level traffic analysis haiphen network trace --target <target>
Knowledge Graph Entity relationships & topology haiphen graph query --q <query>
Risk Analysis Scenario modeling & Monte Carlo haiphen risk assess --scenario <scenario>
Causal Chain Root-cause & event chain analysis haiphen causal analyze --events <file>
Supply Chain Intel Supplier risk scoring & alerts haiphen supply assess --supplier <name>
